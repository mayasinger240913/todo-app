# אפליקציית מטלות לילדים ⭐🧹

אתר שעובד בטלפון: ילדים מבצעים מטלות בית, **מצלמים הוכחה בזמן אמת**,
ומקבלים נקודות. את הנקודות אפשר להמיר לפרסים שההורים מגדירים.
ההורים שולטים על הכל — מאשרים מטלות, מגדירים פרסים, ומאשרים בקשות.

## איך מריצים? (פעם אחת)

1. פותחים טרמינל (PowerShell) בתיקייה הזו.
2. מתקינים את הספרייה הדרושה:
   ```
   pip install -r requirements.txt
   ```
3. מריצים את השרת:
   ```
   python app.py
   ```
4. פותחים בדפדפן: **http://localhost:5000**

## כניסה ראשונה

נוצרים אוטומטית שני משתמשים לדוגמה:

| מי    | שם    | קוד  |
|-------|-------|------|
| הורה  | הורה  | 1234 |
| ילד   | דני   | 1111 |

> ההורה יכול להוסיף עוד ילדים מתוך הטאב "👧 ילדים".

## איך זה עובד?

- **ילד/ה:** בוחר מטלה → לוחץ "עשיתי 📸" → המצלמה נפתחת → מצלם → שולח לאישור.
  אי אפשר להעלות תמונה מהגלריה — רק צילום חי באותו הרגע.
- **הורה:** בטאב "✅ לאישור" רואה את התמונה, מאשר (הנקודות נוספות) או דוחה.
- **פרסים:** ההורה מגדיר פרסים ומחיר בנקודות. הילד מבקש, ההורה מאשר,
  והנקודות **יורדות** כדי שלא ישתמשו בהן שוב.

## להיכנס מהטלפון האמיתי 📱

1. ודאו שהטלפון והמחשב על אותה רשת Wi-Fi.
2. בדקו את כתובת ה-IP של המחשב (ב-PowerShell: `ipconfig` → השדה IPv4).
3. בטלפון פתחו: `http://הכתובת-של-המחשב:5000` (למשל `http://192.168.1.20:5000`).

> ⚠️ המצלמה בדפדפן עובדת רק ב-`localhost` או בכתובת מאובטחת (https).
> מהמחשב עצמו (localhost) זה תמיד יעבוד. מהטלפון ייתכן שתצטרכי הגדרה נוספת בהמשך.

## קבצים בפרויקט

- `app.py` — השרת (Python/Flask) וכל ה-API.
- `templates/index.html` — מבנה הדף.
- `static/style.css` — העיצוב.
- `static/app.js` — כל הלוגיקה בצד הדפדפן.
- `chores.db` — מסד הנתונים (נוצר אוטומטית בהרצה הראשונה).
- `uploads/` — תמונות ההוכחה (נוצר אוטומטית).

## איך מתחילים מאפס?

אם רוצים לאפס הכל — פשוט מוחקים את הקובץ `chores.db` ואת התיקייה `uploads/`,
ומריצים שוב את `python app.py`.
